# 🎲 프리미엄 랜덤 좌석 배정 프로그램

귀엽고 깔끔한 디자인의 학생 좌석 무작위 배정 프로그램입니다.
언제 어디서든 웹 브라우저만 있으면 사용할 수 있도록 제작되었습니다.

## ✨ 주요 기능
- **엑셀 업로드**: 학급 명렬 파일(.xlsx)을 업로드하여 사용 가능
- **자동 배치**: 학생과 좌석을 선택하여 랜덤하게 배치 (애니메이션 효과 포함)
- **드래그 앤 드롭**: 수동으로 학생 위치를 자유롭게 조정
- **시점 전환**: 교사 시점과 학생 시점을 전환하여 확인 가능
- **이미지 저장**: 배치 결과를 날짜와 함께 고화질 이미지로 저장
